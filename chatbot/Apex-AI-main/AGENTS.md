# Agent Rules & Workflows

**Purpose:** Define agent behaviors, workflows, and rules for interaction within Project Brain.

## Development Workflow
Before Every Task, Agents must:
1. Read `brain/master-memory.md`
2. Read `brain/architecture.md`
3. Read `brain/patterns.md`
4. Read `brain/decisions.md`

During Task:
1. Create plan (`implementation-plan.md`)
2. Implement feature
3. Run tests
4. Review output

After Task (Memory Agent):
1. Record new patterns
2. Record decisions
3. Record mistakes fixed
4. Update compressed memory
