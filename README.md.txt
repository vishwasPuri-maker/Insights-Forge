# InsightForge_v1 Integration Workspace

This workspace contains multiple independently developed modules that will be integrated into a single production-ready application.

## Modules

backend/
Main application backend.

frontend/
Main React frontend.

chatbot/
Final Apex AI chatbot implementation.

scraper/
Production data scraping service.

scraper-reference/
Reference/prototype scraper implementation. Use only for comparison or missing ideas. Do not treat it as the production scraper.

## Goal

Analyze all modules, identify dependencies, conflicts and integration points, then produce an integration roadmap before making any code changes.
